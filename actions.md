# Actions

These are the possible actions and their meanings.

| Number | Action |
| ------ | ------ |
| 0      | `do_nothing` |
| 1      | `attack_still` |
| 2      | `loot_still` |
| 3      | `jump_still` |
| 4      | `move_up` |
| 5      | `attack_up` |
| 6      | `loot_up` |
| 7      | `jump_up` |
| 8      | `move_right` |
| 9      | `attack_right` |
| 10     | `loot_right` |
| 11     | `jump_right` |
| 12     | `move_down` |
| 13     | `attack_down` |
| 14     | `loot_down` |
| 15     | `jump_down` |
| 16     | `move_left` |
| 17     | `attack_left` |
| 18     | `loot_left` |
| 19     | `jump_left` |
